450 Blocked by Windows Parental Controls

Descrição

Não oficial: Esse código de status HTTP não é especificado por nenhum RFC.

O código de status de resposta HTTP 450 Blocked by Windows Parental Controls é um código de status HTTP não oficial específico da Microsoft e retornado pelo servidor para indicar que os controles dos pais estão ativados e estão bloqueando o acesso ao recurso solicitado.

Quando o código de status 450 Blocked by Windows Parental Controls é recebido, isso significa que o Windows Parental Controls foi ativado e está negando o acesso ao recurso de destino.
